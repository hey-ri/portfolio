기존 세스코 사이트를 리뉴얼하여 제작하였습니다. 기존 세스코 사이트는 한 페이지에 많은 양의 정보가 담겨 있어서 정보가 더 잘 보이도록 디자인을 구성했습니다. 또 내비게이션 부분의 양은 너무 많고 어려워 재구성하였습니다.
세스코에서 운영하는 렌탈에 대한 정보들도 바로 볼 수 있게 담아놨고 또 세스코 상품에 대한 카테고리 등을 넣어서 정보를 잘 정리하되, 세부적인 내용들은 더 많이 담을 수 있도록 하였습니다.

CODING

  1. 팝업창이 만들어져 있습니다.
  2. 오늘 보지 않기에 대한 쿠키 스크립트가 들어가 있습니다.
  3. flex로 작업했습니다.
  4. 마우스 오버 효과가 있습니다.
  5. slideUp, slideDown으로 네비가 내려옵니다.
  6. swiper로 슬라이드가 돌아갑니다.
  7. select / option을 사용하였습니다.
  8. 푸터부분에 swiper를 사용해 공지사항을 슬라이드가 돌아가도록 하였습니다.

DESIGN

슬라이드 부분에 어떤 솔루션을 이용할 수 있는지 바로 확인 가능하게 하였습니다.

select / option을 이용하여 어떤 카테고리를 이용하고 싶은지 만들어 놓아 바로 예약 가능하도록 하였습니다.

세스코 렌탈 기기 섹션을 놓아 렌탈도 가능하다는 정보를 넣었습니다.

세스코 품목별 상품이라는 섹션을 통해 세스코에 상품을 판다는 것도 알려주었습니다.

베스트 리뷰를 재밌어하는 사용자가 있어 베스트 리뷰도 잘 보이도록 넣어 두었습니다.

공지사항은 자동으로 돌아가게 loop를 이용해 사용자가 움직이지 않아도 작동되도록 하였습니다.
