Main Page Detail

메인 사이트를 보시려면 화면을 클릭해주세요

CODING

  1. flex로 작업했습니다.
  2. 마우스 오버 효과
  3. fixed 효과를 사용해 화면사이드에 배너 창을 만들었습니다.
  4. 화면 하단의 top 버튼은 juqery-scrollTop을 활용해 만들었습니다.

DESIGN

메인 슬라이드 부분에 상담 바로가기를 넣어 상담 예약을 바로 할 수 있습니다.

웹 페이지 우측에 사이드 배너를 넣어 필요한 정보를 바로 얻을 수 있도록 하였습니다.

상담사 선생님들을 확인할 수 있는 카테고리를 넣어서 간단하게 이미지로 파악 할 수 있도록 하였습니다.

상담 카테고리로 원하는 상담 분야가 있다면 바로 예약할 수 있도록 하였습니다.

이용 안내 배너를 넣어 메인 페이에서 내비게이션을 타고 들어가지 않아도 이용 내역을 확인할 수 있게 하였습니다.

그 밑에는 상담 분야를 나눠 그 상담 예약을 할 수 있게 하였습니다.

    2. form태그를 활용하여 표를 구현하였습니다.
    3. selcet/ option 태그 이용하여 상담 신청이 가능한 표를 만들었습니다.

    GNB를 크게 만들어 원하는 상담 분야가 아니면 바로 다른 상담 분야로 넘어가게 만들어 놓았습니다. 그 밑으로는 페이지 대한 정보를 넣어놓았습니다.

    상담 과정과 구성을 표로 만들어 놓았습니다.

    상담 기간과 상담 효과 그 밑으로는 이 구성이 마음에 들었다면 바로 예약할 수 있도록 구성되어 있습니다.

  • 서브 사이트 이용안내를 보시려면 화면을 클릭해주세요

    CODING

    1. 마우스 오버효과
    2. select/option 툴을 이용한 상담신청 표를 만들었습니다.

    DESIGN

    운영방법 & 예약과 관련된 페이지는 상담센터의 이용 시간과 대략적인 비용을 나타내어 사용자가 접근하기 편리하게 하였습니다.

  • 서브 사이트 로그인을 보시려면 화면을 클릭해주세요

    CODING

    1. form, fieldset, legend, input, label 태그를 이용하여 로그인 페이지를 만들었습니다.

    DESIGN

    로그인 페이지는 회원가입을 위한 귀찮음을 최소화하기 위해 간편 로그인을 구글, 네이버, 카카오로 할 수 있도록 하였습니다.
    자동로그인으로 홈페이지에 들어가 매번 로그인 하는 것이 번거롭다고 느끼는 사용자들에게 편의성을 더하여 주었고
    아이디 찾기, 비밀번호 찾기를 이용하여 아이디 비밀번호를 찾을 수 있게 하였습니다.

  • 서브 사이트 회원가입 약관동의을 보시려면 화면을 클릭해주세요

    CODING

    1. 전체 동의를 누르면 전체 동의가 되는 스크립트를 만들었습니다.
    2. 동의를 하면 다음 페이지로 넘어갑니다.
    3. form , input, label을 사용 하였고 input에 onClick 속성이 들어가 있습니다.

    DESIGN

    동의를 누르게 하기 위해 오른쪽에 버튼을 배치하고 취소보다 더 높은 채도를 활용하였습니다.

  • 서브 사이트 회원가입을 보시려면 화면을 클릭해주세요

    CODING

    1. 필수 요소를 입력하지 않으면 넘어가지 않도록 스크립트를 제작하였습니다.
    2. 입력값이 올바르지 않으면 넘어가지 않도록 스크립트를 제작하였습니다.
    3. 입력을 잘 하고 회원가입을 누르면 성공된 페이지로 넘어갑니다.
    4. type을 이용하여 알맞게 태그를 사용합니다.
    5. form, fieldset, legend, input, label 태그를 이용하여 회원가입 페이지를 만들었습니다.
    6. rado 버튼을 사용하여 성별을 나타낼 수 있도록 하였습니다.

    DESIGN

    회원가입 페이지에는 최소한의 정말 필요한 정보들만 입력할 수 있게 하였습니다.
